Jive Messenger @version@

Jive Messenger provides comprehensive group chat and instant messaging (IM) services using the XMPP protocol.

Documentation:

  • Installation Guide - How to install Jive Messenger as a standalone application or into an existing application server.
  • Database Installation Guide - How to setup your database for use with Jive Messenger.
  • SSL Guide - A guide to setting up Messenger's SSL secure socket support.
  • LDAP Guide - A guide to setting up Jive Messenger to work with LDAP user stores.

Developer Documentation:

  • JavaDocs - Jive Messenger API documentation.
  • Plugin Guide - A guide to writing and installing plugins for Jive Messenger.
  • Database Schema Guide - A tour of the Jive Messenger database schema for developers and database administrators.

An active support community for Jive Messenger is available at http://www.jivesoftware.org/forums/.
